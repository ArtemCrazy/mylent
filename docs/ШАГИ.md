# Что делать сейчас — по шагам

## Если хотите просто работать с проектом на компьютере

1. **Откройте папку проекта** в Cursor: `c:\Users\CrazyStudio\OneDrive\Документы\Projects\mylent`

2. **Запустите бэкенд.**  
   Откройте терминал (в Cursor или отдельно), перейдите в папку backend и запустите сервер:
   ```powershell
   cd "c:\Users\CrazyStudio\OneDrive\Документы\Projects\mylent\backend"
   .\.venv\Scripts\Activate.ps1
   uvicorn app.main:app --reload --port 8000
   ```
   Оставьте это окно открытым.

3. **Запустите фронт.**  
   Откройте **второй** терминал:
   ```powershell
   cd "c:\Users\CrazyStudio\OneDrive\Документы\Projects\mylent\frontend"
   npm run dev
   ```
   Оставьте это окно открытым.

4. **Откройте в браузере:** http://localhost:3000  
   Войдите (если уже создавали пользователя — свой логин и пароль; иначе сначала создайте пользователя через скрипт в backend).

5. **Чтобы посты из Telegram появлялись в ленте**, в **третьем** терминале один раз выполните импорт, потом включите реал-тайм:
   ```powershell
   cd "c:\Users\CrazyStudio\OneDrive\Документы\Projects\mylent\backend"
   .\.venv\Scripts\python.exe -m scripts.telegram_sync
   ```
   (телефон и код — один раз). Потом:
   ```powershell
   .\.venv\Scripts\python.exe -m scripts.telegram_realtime
   ```
   Оставьте этот процесс включённым — тогда новые посты будут подтягиваться сами.

## Если хотите выложить сайт на ваш VPS (интернет)

1. **Убедитесь, что последние правки на GitHub.**  
   В терминале из папки проекта:
   ```powershell
   cd "c:\Users\CrazyStudio\OneDrive\Документы\Projects\mylent"
   .\scripts\push.ps1
   ```
   Если напишет «No changes» — уже всё залито.

2. **Подключитесь к VPS.**  
   На вашем компьютере откройте PowerShell и выполните:
   ```powershell
   ssh root@155.212.219.106
   ```
   Введите пароль от сервера (тот, что дал хостинг).

3. **На VPS выполните одну команду** (скопируйте целиком и вставьте в терминал):
   ```bash
   apt-get update && apt-get install -y git && git clone https://github.com/ArtemCrazy/mylent.git /root/mylent && bash /root/mylent/scripts/server-setup.sh
   ```
   Подождите, пока всё установится и запустится (несколько минут).

4. **Откройте в браузере:** http://155.212.219.106:3000  
   Должен открыться ваш сайт.

5. **На VPS создайте пользователя для входа** (логин и пароль для сайта):
   ```bash
   cd /root/mylent/docker
   docker compose exec backend python -m scripts.create_user
   ```
   Введите логин и пароль, которыми будете входить на сайт.

6. **Один раз привяжите Telegram** (чтобы импортировать каналы):
   ```bash
   docker compose exec -it backend python -m scripts.telegram_sync
   ```
   Введите номер телефона и код из Telegram.

7. **Включите подтягивание постов раз в 10 минут:**
   ```bash
   docker compose exec -d backend python -m scripts.telegram_sync_loop
   ```

После этого сайт в интернете будет работать по адресу http://155.212.219.106:3000.
