# PRD: Персональная AI-лента (MyLent)

## Продукт
Веб-сервис для одного пользователя: агрегация публикаций из Telegram-каналов в единую ленту с AI-обработкой (summary, теги, важность), поиском, фильтрами, избранным и дайджестами.

## Цель
Убрать разрозненное потребление контента: одна лента вместо десятков каналов, быстрые фильтры, сохранение важного, AI-сводки и подборка «полезно для студии».

## ЦА (MVP)
Один пользователь — руководитель веб-студии. Нужно: быстро просматривать каналы, выделять полезное по бизнесу/AI/маркетингу/управлению, сохранять идеи и кейсы, получать компактные сводки.

## Ключевые возможности MVP
- Подключение Telegram-источников, автоимпорт постов
- Единая лента с карточками, сортировка и фильтры
- Поиск по тексту, тегам, источнику
- Избранное, скрытие, архив
- AI: summary, теги, тема, важность, релевантность для бизнеса
- Раздел «Полезно для студии»
- Дайджесты (ручная/авто генерация)
- Управление источниками и настройки

## Формат
Веб-приложение, desktop-first, тёмная тема, карточная лента, минималистичный UI.

## Критерии успеха MVP
Пользователь подключает каналы → посты импортируются → всё в одной ленте → поиск и фильтры работают → избранное и скрытие → AI-summary и теги у большинства постов → есть подборка «полезно для студии» и дайджест → интерфейс не выглядит сырым внутренним тулом.
