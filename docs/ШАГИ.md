## Если хотите выложить сайт на ваш VPS (интернет)

**На VPS трогаем только MyLent:** папка `/root/mylent`, порты **3001** (сайт), **8001** (API), **5433** (Postgres). Парсер и другие проекты на сервере не трогаем — они работают на своих портах (3000, 8000 и т.д.).

**Один раз** настройте деплой через GitHub Action (тогда при каждом пуше в `main` сайт будет обновляться сам, без вашего участия):

1. **Один раз поднимите проект на VPS** (если ещё не поднимали): зайдите по SSH на сервер и выполните:
   ```bash
   apt-get update && apt-get install -y git && git clone https://github.com/ArtemCrazy/mylent.git /root/mylent && bash /root/mylent/scripts/server-setup.sh
   ```
   Дальше создайте пользователя и привяжите Telegram (шаги 5–7 ниже) — это тоже один раз.

2. **Один раз добавьте секреты в GitHub:**  
   Репозиторий → **Settings** → **Secrets and variables** → **Actions** → **New repository secret**. Создайте три секрета:
   - `DEPLOY_HOST` — IP или домен сервера, например `155.212.219.106`
   - `DEPLOY_USER` — пользователь SSH, например `root`
   - `SSH_PRIVATE_KEY` — приватный SSH-ключ (содержимое файла, которым вы заходите на сервер). Если ключа нет: на ПК выполните `ssh-keygen -t ed25519 -f deploy_key -N ""`, скопируйте `deploy_key.pub` на сервер в `~/.ssh/authorized_keys`, в секрет положите содержимое файла `deploy_key`.

После этого **ваше участие не нужно**: делаете `.\scripts\push.ps1` (или пушите в `main`) — GitHub Action сам зайдёт на VPS, выполнит `git pull` и `docker compose up -d`.

---

**Если не хотите использовать Action** и будете обновлять сервер вручную:

1. **Убедитесь, что последние правки на GitHub** (`.\scripts\push.ps1`).

2. **Подключитесь к VPS:** `ssh root@155.212.219.106`

3. **На VPS выполните** (скопируйте целиком и вставьте в терминал):
   ```bash
   apt-get update && apt-get install -y git && git clone https://github.com/ArtemCrazy/mylent.git /root/mylent && bash /root/mylent/scripts/server-setup.sh
   ```
   Подождите, пока всё установится и запустится (несколько минут).

4. **Откройте в браузере:** http://155.212.219.106:3001  
   Должен открыться ваш сайт.

5. **На VPS создайте пользователя для входа** (логин и пароль для сайта):
   ```bash
   cd /root/mylent/docker
   docker compose exec backend python -m scripts.create_user
   ```
   Введите логин и пароль, которыми будете входить на сайт.

6. **Один раз привяжите Telegram** (чтобы импортировать каналы):
   ```bash
   docker compose exec -it backend python -m scripts.telegram_sync
   ```
   Введите номер телефона и код из Telegram.

7. **Включите подтягивание постов раз в 10 минут:**
   ```bash
   docker compose exec -d backend python -m scripts.telegram_sync_loop
   ```

После этого сайт в интернете будет работать по адресу http://155.212.219.106:3001.

**Если backend не запускается** (ошибка `address already in use` на порту 8000): на VPS заняты порты 3000/8000 другими проектами (парсер и т.д.). Обновите код и перезапустите **только MyLent** — в репозитории заданы порты 3001 и 8001:
```bash
cd /root/mylent && git pull && cd docker && docker compose -p mylent down && docker compose -p mylent up -d
```
